# meow-lang
### meow is a programming language about cats! (;

---

## Why meow?
- **Powerful**: With meow, you can do *anything* — from *console applications* to entire *operating systems*!
- **Simple**: meow is a really simple language to learn. We have a book you can read to learn it! (:
- **Fast**: meow is a fast, interpretable language powered by **C**!

## Why **NOT** meow? (I'll be honest)
- **Pre-release**: The current version is v0.1 beta, so it is *buggy*...
